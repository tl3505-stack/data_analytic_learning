Part 1 · StatsModels — Q1 to Q12
Q1Read in the data. Split 70/30 using DataFrame.sample(), random_state=617. What percentage of employees in the train sample left the company?
CODE
import pandas as pd
import numpy as np

data = pd.read_csv('employee_attrition.csv')

train = data.sample(frac=0.7, random_state=617)
test  = data.drop(labels=train.index)

pct = round(train['attrition'].mean() * 100, 2)
print(pct)
✅ 答案：20.7
📌 考点：用 DataFrame.sample() 做简单随机抽样，注意是 frac=0.7 而不是 train_size=0.7
Q2Compare employees who left vs stayed based on average age. What is the average age of employees who left?
CODE
avg_age_by_attrition = train.groupby('attrition')['age'].mean()
print(avg_age_by_attrition)

# Or directly:
print(round(train[train['attrition']==1]['age'].mean(), 2))
✅ 答案：48.3
📌 离职员工平均年龄约 48.3 岁，高于未离职员工，提示年龄可能与离职正相关。
Q3Fit model1 = logit('attrition ~ age'). Which conclusion is correct?
CODE
from statsmodels.formula.api import logit

model1 = logit('attrition ~ age', data=train).fit()
print(model1.summary())
✅ 答案：age 的系数为正（≈+0.013），p-value < 0.05 → 年龄越大，离职概率越高，且效果显著
📌 考点：读 summary() 中系数的符号和 P>|z| 值。正系数 + 显著 = 正向影响离职。
Q4Based on model1, what is the predicted probability that the first person in the train sample will leave?
CODE
first_person = train.iloc[[0]][['age']]
prob = model1.predict(first_person)
print(round(prob.values[0], 4))
✅ 答案：0.2077（first person 年龄 = 48）
📌 用 .predict() 传入 DataFrame，注意 iloc[[0]] 保留二维结构。
Q5Based on model1, what is the probability that a 60-year-old will leave?
CODE
age_60 = pd.DataFrame({'age': [60]})
prob_60 = model1.predict(age_60)
print(round(prob_60.values[0], 4))
✅ 答案：0.235
📌 新数据点预测：构造一个只有 age=60 的 DataFrame 传入 .predict()。
Q6Based on model1, if age goes up by one year, what is the percent change in likelihood of leaving?
CODE
import math
pct_change = (math.exp(model1.params['age']) - 1) * 100
print(round(pct_change, 2))
✅ 答案：+1.33%（每增加 1 岁，离职可能性增加约 1.33%）
📌 核心公式：(exp(β) − 1) × 100。系数正 → 正变化 → 离职风险随年龄增加。
Q7Examine attrition rate by gender. Which statement is true?
CODE
attrition_by_gender = train.groupby('gender')['attrition'].mean() * 100
print(attrition_by_gender.round(2))
✅ 答案：Female 离职率 ≈ 24.17%，Male 离职率 ≈ 18.55% → 女性离职率更高
📌 用 groupby + mean() × 100 计算各组离职率。
Q8Fit model2 = logit('attrition ~ gender'). What is the difference in likelihood of male vs female leaving?
CODE
model2 = logit('attrition ~ gender', data=train).fit()
print(model2.summary())

# 男性 vs 女性的 odds ratio（女性为 baseline）
odds_ratio = math.exp(model2.params['gender[T.Male]'])
pct_diff = (odds_ratio - 1) * 100
print(f"Odds ratio: {round(odds_ratio, 4)}")
print(f"% difference: {round(pct_diff, 2)}%")
✅ 答案：Male 系数 ≈ −0.336，Odds ratio ≈ 0.714 → 男性比女性离职可能性低约 28.55%
📌 女性为 baseline（字母排序先），Male 系数为负 → 男性离职风险更低，与 Q7 一致。
Q9Fit model3 = logit('attrition ~ department'). Which statements are true?
CODE
model3 = logit('attrition ~ department', data=train).fit()
print(model3.summary())

# Baseline = Human Resources（字母排序第一）
# R&D 系数, Sales 系数
for k, v in model3.params.items():
    print(f"{k}: coef={round(v,4)}, odds_ratio={round(math.exp(v),4)}")
✅ 答案： • Baseline = Human Resources • R&D 系数 ≈ −0.315（odds ratio ≈ 0.73），R&D 离职率低于 HR（但需看 p-value 是否显著） • Sales 系数 ≈ +0.036（odds ratio ≈ 1.04），Sales 与 HR 差异不显著
📌 StatsModels 自动设定字母排序最小的类别为 baseline。需看每个系数的 p-value 判断显著性。
Q10Fit model4 with all variables. What is the Pseudo R-squared?
CODE
model4 = logit(
    'attrition ~ age + gender + department + years + performance + jobsat + income + distance',
    data=train
).fit()

print(round(model4.prsquared, 4))
# 或从 summary() 中读取 Pseudo R-squ.
✅ 答案：≈ 0.3486
📌 McFadden Pseudo R² = 1 − (LL_fitted / LL_null)。0.35 属于较好的拟合（通常 >0.2 即可接受）。
Q11Based on model4, which variables have an effect on attrition? (Select one or more)
CODE
print(model4.summary())
# 看每个变量的 P>|z| 列，p < 0.05 则显著
✅ 显著变量（p < 0.05）：years、performance、jobsat、gender（Male） ❌ 不显著：income、distance、department（Sales）、age（边缘）
📌 p-value < 0.05 才能认为该变量对 attrition 有显著影响。income 系数极小（≈0），没有实际作用。
Q12Based on model4, what is the impact of a one-unit increase in jobsat on likelihood to leave?
CODE
b_jobsat = model4.params['jobsat']
impact = (math.exp(b_jobsat) - 1) * 100
print(round(impact, 2))
✅ 答案：−35.99%（jobsat 每提高 1 分，离职可能性降低约 36%）
📌 jobsat 系数 ≈ −0.446，负向显著。工作满意度越高，离职风险越低，符合直觉。
Part 2 · Scikit-Learn — Q13 to Q19
Q13Stratified split 80/20, random_state=617. What % of train left?
CODE
from sklearn.model_selection import train_test_split

y = data['attrition']
X = data.drop('attrition', axis=1)

X_train_raw, X_test_raw, y_train, y_test = train_test_split(

✅ 答案：20.52
📌 关键区别：stratify=y 保证 train/test 中 attrition 比例相同；Pandas .sample() 不保证这一点。
Q14Dummy code gender and department (drop_first=True). Reindex test to train columns. What is the mean of dummy coded gender?

print(round(X_train['gender_Male'].mean(), 4))
✅ 答案：0.6012（train 中约 60.12% 为男性）
📌 drop_first=True 后 gender 只保留 gender_Male（Female 为 baseline）。reindex 确保 test 列与 train 完全一致。
Q15Fit LogisticRegression(max_iter=10000). What is the predicted value for the first observation in train?
CODE
from sklearn.linear_model import LogisticRegression

logit_sk = LogisticRegression(max_iter=10000, random_state=617)
logit_sk.fit(X_train, y_train)

y_pred_train = logit_sk.predict(X_train)
print(y_pred_train[0])          # 预测类别：0 或 1

y_pred_proba_train = logit_sk.predict_proba(X_train)
print(y_pred_proba_train[0])    # [P(0), P(1)]
✅ 答案：预测类别 = 0（不会离职），P(attrition=1) ≈ 0.006
📌 .predict() 返回类别，.predict_proba()[:,1] 返回正类概率。默认 threshold=0.5。
Q16What is the Sensitivity (Recall) in the train sample?
CODE
from sklearn.metrics import recall_score
sensitivity_train = recall_score(y_train, y_pred_train)
print(round(sensitivity_train, 4))
✅ 答案：0.4932（即 49.32%）
📌 Sensitivity = Recall = TP / (TP + FN)。较低的 Sensitivity 说明模型漏掉了约一半真正会离职的员工。
Q17What is the Sensitivity in the test sample?
CODE
y_pred_test = logit_sk.predict(X_test)
sensitivity_test = recall_score(y_test, y_pred_test)
print(round(sensitivity_test, 4))
✅ 答案：0.5234（即 52.34%）
📌 test Sensitivity 略高于 train，说明模型没有过拟合，泛化能力正常。
Q18What is the AUC for the train sample?
CODE
from sklearn.metrics import roc_auc_score
y_pred_proba_train = logit_sk.predict_proba(X_train)
auc_train = roc_auc_score(y_train, y_pred_proba_train[:,1])
print(round(auc_train, 4))
✅ 答案：0.8789
📌 AUC 接近 0.88，说明模型区分离职/留职能力很强（AUC=0.5 随机，=1.0 完美）。
Q19What is the AUC for the test sample?
CODE
y_pred_proba_test = logit_sk.predict_proba(X_test)
auc_test = roc_auc_score(y_test, y_pred_proba_test[:,1])
print(round(auc_test, 4))
✅ 答案：0.8833
📌 test AUC ≈ train AUC，模型泛化良好。Sensitivity 低但 AUC 高 → 概率排序好，但 threshold=0.5 不是最优截断点。

B7[Conceptual] The dataset has 95% non-default and 5% default. A naive model that always predicts "no default" has 95% accuracy. Explain why this is misleading and which metric should be prioritized.
概念题答案框架
# 这是一个 Class Imbalance 问题。
# 解释：
# 1. Accuracy 被多数类主导 → 95% 准确但完全没有预测价值
# 2. 关键错误是 False Negatives（把违约客户预测为不违约）
#    → 银行会蒙受实际损失
# 3. 应优先看：
#    - Sensitivity/Recall：真正违约中被识别的比例
#    - AUC：不依赖阈值，综合衡量模型区分能力
#    - Precision-Recall curve：在 imbalanced 数据中比 ROC 更信息量大
📌 这类概念题在期末考试中经常出现！记住：Imbalance → Accuracy 失效 → 用 Sensitivity/AUC。
